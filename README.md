# Enhanced Face Tracer
Enhanced implementation of FaceTracer for source identity tracing.

## Features
- Source identity extraction from face-swapped content
- ViT backbone with AAMSoftmax enhancement
- Identity consistency analysis
- Cross-frame matching

## Project Structure
- main.py - Main pipeline
- models/ - Model architectures
- utils/ - Utility functions
- scripts/ - Helper scripts
- checkpoints/ - Model weights (download separately)
- input_videos/ - Input data (add your data)
- output_videos/ - Output results
